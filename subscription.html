Integration with
the controller
The keypass.cloud controller is intended to be used with a
cloud control panel. Internet access isprovided via Wi-Fi or
Ethernet. If there is a problem connecting to the internet the
controller will work offline saving all the logs autonomously.
The data will be synchronised when the controllerwill be
connected to the cloud again.

FAQ (Frequently Asked Questions)

1. Cloud and internet are not stable. If the Internet stops working the doors won’t open?
That was the first issue we have solved. The controller is a small computer as itself and itstores settings and log events locally.
As soon as the Internet access is stable again, the controller synchronises all the settings and logs with the cloud.
